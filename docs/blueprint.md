# **App Name**: Jal Suraksha

## Core Features:

- Animated Background: Dynamically renders moving waves using different shades of blue. The background animation runs smoothly.
- Interactive Elements: Interactive UI elements designed on top of the animated background for user interaction.
- Navigation Menu: A standard navigation menu to allow users to easily move around the app
- AI Chatbot for Symptom Detection: Use an AI chatbot to detect symptoms and suggest possible diseases and cures.

## Style Guidelines:

- Primary color: Deep blue (#3498DB) evoking depth and serenity.
- Background color: Light blue (#EBF5FB) as a desaturated hue of the primary, creating a calm atmosphere.
- Accent color: Soft lavender (#A9CCE3) adding a subtle contrast.
- Body and headline font: 'Inter', a grotesque-style sans-serif, for a modern and neutral appearance.
- Fluid, continuous wave motion in the background with subtle parallax effect.
- Minimalist layout to ensure focus on the animated background and content.
